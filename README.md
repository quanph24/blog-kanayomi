# blog-kanayomi
https://blog.kanayomi.com/

### Install Hexo
```
$ npm install -g hexo-cli
```

### Install blog
```
$ cd <personal-blog>
$ npm install
```

### Run server
```
$ hexo server
```

### generate
```
$ hexo generate
```

### clean
```
$ hexo clean
```

### Deploy
```
$ hexo generate
$ gcloud app deploy
```
